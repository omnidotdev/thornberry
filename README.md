# 🍓 Thornberry

**Omni Sigil Thornberry** is a component library and implementation of the Omni Sigil design system using [Tailwind](https://tailwindcss.com) and [shadcn](https://ui.shadcn.com).

## Local Development

Run `tilt up`, or:

```sh
bun install
```

```sh
bun dev
```
