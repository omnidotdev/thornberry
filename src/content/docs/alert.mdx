## Usage

For simplicity, we provide a pre-built component that you can use out of the box.

```tsx
import { Alert } from "@/components/ui/alert";
```

```tsx
<Alert title="Alert" description="This is an alert component for important messages." />
```

<br />

### Primitives

If you need more control over the alert component, you can use the primitives.

```tsx
import { AlertRoot, AlertIcon, AlertTitle, AlertDescription } from "@/components/ui/alert";
```

```tsx
<AlertRoot>
  <AlertIcon>
    <YourIcon />
  </AlertIcon>
  <AlertTitle>Title</AlertTitle>
  <AlertDescription>Description</AlertDescription>
</AlertRoot>
```
