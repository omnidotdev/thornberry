### Examples

Let's see how to use the `Badge` component. Let's start with a basic example:

```tsx
import { Badge } from "@/components/ui/badge"

export const BasicBadge = () => (
  <Badge>Default Badge</Badge>
)
```

The Badge component comes with different variants that you can use to represent different styles:

```tsx
import { Badge } from "@/components/ui/badge"

export const BadgeVariants = () => (
  <>
    <Badge>Solid</Badge>
    
    <Badge variant="subtle">Subtle</Badge>
    
    <Badge variant="outline">Outline</Badge>
  </>
)
```

You can also use different sizes for your badges:

```tsx
import { Badge } from "@/components/ui/badge"

export const BadgeSizes = () => (
  <>
    <Badge size="sm">Small</Badge>
    
    <Badge>Medium</Badge>
    
    <Badge size="lg">Large</Badge>
  </>
)
```
