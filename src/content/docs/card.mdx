## Usage

For simplicity, we provide a pre-built component that you can use out of the box.

```tsx
import { Card } from "@/components/ui/card";
```

```tsx
<Card title="Card Title" description="Card Description" footer={<Button>Save</Button>}>
  This is the content area of the card, where the main information is displayed.
</Card>
```

### Primitives

If you need more control over the card component, you can use the primitives.

```tsx
import {
  CardRoot,
  CardHeader,
  CardTitle,
  CardDescription,
  CardContent,
  CardFooter
} from "@/components/ui/card";
```

```tsx
<CardRoot>
  <CardHeader>
    <CardTitle>Card Title</CardTitle>
    <CardDescription>Card Description</CardDescription>
  </CardHeader>
  <CardContent>
    This is the content area of the card, where the main information is displayed.
  </CardContent>
  <CardFooter>
    <Button>Save</Button>
  </CardFooter>
</CardRoot>
```
