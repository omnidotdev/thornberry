## Fixed Value

<TabsRoot value="code">
  <TabsList className="grid w-full grid-cols-2">
    <TabsTrigger value="code">Code</TabsTrigger>
    <TabsTrigger value="preview">Preview</TabsTrigger>
  </TabsList>
  <TabsContent value="code">
    <div className="p-4 rounded-md bg-muted font-mono text-sm">
      const greeting = "Hello, world!";
      <br />
      console.log(greeting);
    </div>
  </TabsContent>
  <TabsContent value="preview">
    <div className="p-4 rounded-md border flex items-center justify-center">
      <div className="text-xl">Hello, world!</div>
    </div>
  </TabsContent>
</TabsRoot>

## Primitives

<TabsRoot defaultValue="primitives">
  <TabsList>
    <TabsTrigger value="primitives">Primitives</TabsTrigger>
    <TabsTrigger value="usage">Usage</TabsTrigger>
    <TabsTrigger value="styling">Styling</TabsTrigger>
  </TabsList>
  <TabsContent value="primitives">
    <div className="p-4 rounded-md border">
      <div className="text-sm">
        Tabs are built using primitive components that you can compose to create custom tab interfaces.
        The main primitives are <code>TabsRoot</code>, <code>TabsList</code>, <code>TabsTrigger</code>, and <code>TabsContent</code>.
      </div>
    </div>
  </TabsContent>
  <TabsContent value="usage">
    <div className="p-4 rounded-md border">
      <div className="text-sm">
        You can use the simplified <code>Tabs</code> component by passing an array of tab objects, or build more complex
        tab interfaces using the primitive components directly.
      </div>
    </div>
  </TabsContent>
  <TabsContent value="styling">
    <div className="p-4 rounded-md border">
      <div className="text-sm">
        All tab components accept a <code>className</code> prop for custom styling using Tailwind CSS or other CSS classes.
        You can also style tabs using data attributes for different states.
      </div>
    </div>
  </TabsContent>
</TabsRoot>
