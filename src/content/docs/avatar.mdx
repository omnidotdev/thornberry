## Usage

For simplicity, we provide a pre-built component that you can use out of the box.

```tsx
import { Avatar } from "@/components/ui/avatar";
```

```tsx
<Avatar src="/img/elon-musk.jpg" alt="Elon Musk" fallback="EM" />
```

<br />

### Primitives

If you need more control over the avatar component, you can use the primitives.

```tsx
import { AvatarRoot, AvatarFallback, AvatarImage } from "@/components/ui/avatar";
```

```tsx
<AvatarRoot>
  <AvatarImage src="/path-to-image.jpg" alt="User name" />
  <AvatarFallback>UN</AvatarFallback>
</AvatarRoot>
```
