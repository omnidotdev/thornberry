### Examples

Let's see how to use the `Avatar` component. Let's start with a basic example:

```tsx
import { Avatar } from "@/components/ui/avatar"

export const BasicAvatar = () => (
  <Avatar name="Elon Musk" imageSrc="/img/elon-musk.jpg" />
)
```

If the user's name is not known ahead of time, you can use the `fallback` prop to provide a fallback icon.

```tsx
import { Avatar } from "@/components/ui/avatar"

export const FallbackAvatar = () => (
  <Avatar fallback={<UserIcon />} imageSrc="/img/elon-musk.jpg" />
)
```

Thornberry also ships primitives for more granular control:

```tsx
import { AvatarFallback, AvatarImage, AvatarRoot } from "@/components/ui/avatar"

export const Primatives = () => (
  <AvatarRoot>
    <AvatarFallback>EM</AvatarFallback>
    <AvatarImage src="/img/elon-musk.jpg" />
  </AvatarRoot>
)
```
